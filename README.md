# Backend-сервис автоматизации закупок 🛒

### Backend-приложение для автоматизации процесса закупки товаров в розничной сети через REST API.

## Статус выполнения проекта ✅
| Этап | Задача | Статус |
|:---|:---|---:|
| **1** | Создание и настройка проекта. | ✅ **Done** |
| **2** | Проработка моделей данных. | ⏳ **В работе** |
| **3** | Реализация импорта товаров.| ⏳ План |
| **4** | Реализация API views. | ⏳ План |
| **5** | Полностью готовый backend. | ⏳ План |

## 🖥 Техническое окружение
* **ОС**: Ubuntu LTS (в среде VirtualBox/WSL2).
* **Python:** >= 3.10.
* **Виртуальное окружение:** venv.
* **IDE:** VS Code / PyCharm.
* **Особенности среды (Shared Folders)**

#### * Проект разрабатывается на Windows-хосте с монтированием папки через /media/sf_linux_share. Для корректной работы Python/Django внутри Linux-среды используется изолированное виртуальное окружение (venv), что исключает ошибки символических ссылок файловой системы NTFS.

## 🛠 Инструкция по запуску (Development)
### Шаг 1. Подготовка зависимостей
```text 
cd /media/sf_linux_share/diploma_project
source env/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```
### Шаг 2. Инициализация БД и Админ-панели
```bash
python manage.py migrate
python manage.py createsuperuser
```

### Шаг 3. Запуск сервера
```bash
python manage.py runserver 0.0.0.0:8000
# Приложение доступно по адресу: http://127.0.0.1:8000/admin/
```
